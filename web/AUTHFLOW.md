## The user enters their credentials in their web browser.

## The browser sends the credentials to your server.

## Your server uses the Server SDK to authenticate the user with Appwrite.

## If the authentication is successful, your server sends a session cookie to the client's web browser.

## The client's web browser sends the session cookie to your server with subsequent request.

## Your server uses the session cookie to make authenticated requests to Appwrite on behalf of the client.
